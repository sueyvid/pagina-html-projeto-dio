Estrutura básica do HTML

A estrutura básica do HTML contém 3 tags principais:

  • tag html
  • tag head
  • tag body

A tag html engloba as outras duas, head e body. Em head, coloca-se comandos de configurações da página, enquanto que em body, coloca-se o conteúdo propriamente dito da página.

Falando sobre tags

As tags são os elementos responsáveis por marcar o texto HTML.

A maioria desses elementos são compostos por:

  • tag de abertura
  • conteúdo
  • tag de fechamento

Atributos das tags

As tags podem conter atributos, que podem modificar o comportamento e/ou propriedades dessas tags.

Textos

Algumas tags que estão relacionadas a textos são:

  • tag p
  • tag h1 ... h6
  • tag blockquote

Essas tags definem tipos de textos diferentes. h1 ... h6 para títulos e subtítulos, p para parágrafos e blockquote para citações.

Algumas tags são responsáveis por modificar características desses textos formatando-os, como:

  • tag strong
  • tag i
  • tag mark

A tag strong torna o texto em negrito, a tag i em itálico, e a tag mark grifa o texto.

Listas

As listas podem ser ordenadas ou não-ordenadas. Para isso usa-se as tags ol e ul respectivamente. Além disso para cada item da lista deve-se envolvê-lo pela tag li.

Exemplo de lista ordenada:

  1. Casa
  2. Carro
  3. Casamento

Exemplo de lista não-ordenada:

  1. Sair para passear
  2. Comer pipoca
  3. Varrer a casa

Links

Os links são responsáveis por interligar as páginas da internet.

Há 3 opções de links para se utilizar em uma página HTML:

  • Linkar uma página externa
  • Linkar uma página do próprio website
  • Linkar a uma seção específica da prŕopria página HTML

A tag responsável por criar os links é a tag a, o atributo que contém o link e href.

Para linkar uma página externa ou interna basta passar o caminho para o atributo href.

Para linkar uma seção, o caminho deve ser um id de uma tag existente na página atual.
